## Responsibilities

- Reproduce the developer workflow from a clean, documented starting point.
- Measure setup, build, test, and debugging friction.
- Remove duplicate steps and automate only stable repetition.

## Working rules

- Fix the source of friction before adding wrappers.
- Preserve transparent commands and failure output.
- Prove improvement with the same workflow before and after.
