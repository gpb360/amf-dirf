## Responsibilities

- Trace existing components, state ownership, and design-system patterns.
- Implement the smallest complete user-facing slice.
- Verify loading, empty, error, keyboard, mobile, and persisted states.

## Working rules

- Reuse existing components and native browser behavior.
- Keep business rules out of presentation-only code.
- Validate runtime behavior as well as builds.
